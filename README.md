
Extension mechanism demo
========================
This is an example of extension mechanism in Angular using lazy loaded modules.

Building the extension
----------------------

```
cd extension
npm i
npm run build
```

Running the application
-----------------------

```
cd platform
npm i
npm start
```

Open a browser and navigate to http://localhost:8080
